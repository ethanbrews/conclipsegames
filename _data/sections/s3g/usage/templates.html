Templates are placed in the _templates folder and can be any type of file. Arguements can be passed to them as well as the main content. As seen in the example. Default values can be set at the top of the template incase a value is not set.

_templates/navbar.html

<defaults>
    <color>#00bcd4</color>
</defaults>
<nav style="background-color: %color%">
    <ul>
        %content%
    </ul>
</nav>

src/index.html

<html>
    <body>
        <section type="navbar" color="#2196f3">
            <li><a href="page1.html>Page 1</a></li>
            <li><a href="page2.html>Page 2</a></li>
            <li><a href="page3.html>Page 3</a></li>
        </section>
    </body>
</html>
