Data files are stored in the _data folder and by default only .json files are supported. If .html, .css or .js files are found, the contents of those files will also be read. Variables are accessed by placing the name of the variable in %percent% symbols. An example is shown below.

File Structure

|--_data/
|  |--site.json
|  |--fonts/
|  |  |--slab.html
|
|--src/
|  |--index.html

site.json

{
    "homepage": {
        "title": "Page Title"
    }
}

slab.html

<link href='https://fonts.googleapis.com/css?family=Roboto+Slab' rel='stylesheet'>
<style>.slab {font-family: 'Roboto Slab', serif;}</style>

index.html

<html>
    <head>
        <title>%site.homepage.title%</title>
        %fonts.slab%
    </head>
    <body>
        <h1 class="slab">%site.homepage.title%</h1>
    </body>
</html>

Output

<html>
    <head>
        <title>Page Title</title>
        
        <link href='https://fonts.googleapis.com/css?family=Roboto+Slab' rel='stylesheet'>
        <style>.slab {font-family: 'Roboto Slab', serif;}</style>
        
    </head>
    <body>
        <h1 class="slab">Page Title</h1>
    </body>
</html>
