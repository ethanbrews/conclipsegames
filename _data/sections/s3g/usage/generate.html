To run the generator, open a command window in the _s3g directory then run python start.py <args> The generator will then generate the site using any plugins provided.

Argument Default Description
--log-level INFO The log level of the program output.
--iter-limit 60 Maximum number of times a plugin can process a page. Stops a page being caught in an infinite loop.
--working-directory ./ The working directory of S3G.
--server true Should the built in server should be started
--server-ip localhost The IP of the built in server
--server-port 8000 The port of the built in server
--server-refresh true Should the site regenerate when a change is detected in the src folder.
