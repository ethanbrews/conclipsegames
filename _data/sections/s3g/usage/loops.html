For loops can loop over each variable in an array (stored in _data) or over a range of numbers. An example is shown below.

pages.json

{
    "pages": [
        {"title": "Example Page"},
        {"title": "Second Example"},
        {"title": "Another Page"}
    ]
}

Input

<ul>
    <for item in pages.pages>
        <li>%item.title%</li>
    </for>
</ul>

Output

<ul>
    <li>Example Page</li>
    <li>Second Example</li>
    <li>Another Page</li>
</ul>

For loops can also loop over a range of numbers.

Input

<for i from 1 to 5>
    <p>%i%</p>
</for>

Output

<p>1</p>
<p>2</p>
<p>3</p>
<p>4</p>
<p>5</p>

The step can also be changed as shown in the example below.

Input

<for i from 5 to 25 step 5>
    <p>%i%</p>
</for>

Output

<p>5</p>
<p>10</p>
<p>15</p>
<p>20</p>
<p>25</p>
